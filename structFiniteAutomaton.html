Detailed Description

Struct of a finite automaton.

Parameters
alphabet: alphabet of the automaton
initialState: initial state of the automaton
numberOfStates: number of states of the automaton
states: array of states of the automaton
transition: Transition matrix of the automaton, the first transition table corresponds to the first state, the second to the second state, and the last to the index of the letter in the alphabet.
